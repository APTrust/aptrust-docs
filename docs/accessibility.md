# Accessibility Statement

## Overview

**Academic Preservation Trust is committed to providing a website that is accessible to everyone**. To further this commitment, we are using the [Web Content Accessibility Guidelines 2.2, Level AA](https://www.w3.org/TR/WCAG22/#conformance-reqs), which provide standards for making web content more accessible to all visitors, including people with disabilities.

## Continuous Monitoring of Content

The website undergoes regular testing with current software and methodologies to maintain accessible content. Testing includes:

- Assistive technology and screen reader software
- Keyboard-only navigation functionality
- Color contrast analysis
- Magnification software
- Usability testing

The organization aims to surpass minimum accessibility standards by ensuring practical usability. Issues are addressed promptly, with assistance provided when immediate solutions aren't available.
